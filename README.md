### GraphQL API와 REST API는 서로 다르다
REST API : URL 사용하여 JSON을 얻고 얻은 JSON으로 원하는 것을 한다.         
GraphQL API : query를 이용하여 데이터를 얻는다. (query는 Axios의 fetch와 함께 POST request로 보내진다.)
        
### Apollo Client
GraphQL API를 호출하기 위해 사용되는 라이브러리이다.
